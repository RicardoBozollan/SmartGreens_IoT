# STEP 1: Connect the sensors and relay to the ESP32.

### Materials Needed:
- ESP32
- Single-board computer (SBC)
- Light Dependent Resistor (LDR)
- Resistor 10K ohms
- Relay Module
- Soil Moisture Sensor
- 1 Channel Relay Module 12V
